# Codex 全局上下文提示词

你正在开发一个项目：**AI 图像分类虚拟实验室：从样本到模型的可视化训练实验**。

项目定位：

这是一个面向高校非计算机专业学生的 AI 图像分类虚拟实验作品。项目不是大型教育平台，不优先做评论、收藏、教师后台、管理员后台、复杂社区。项目核心是让学生通过零代码可视化实验理解图像分类模型训练过程。

核心实验闭环：

1. 实验目标。
2. 实验原理。
3. 数据集准备。
4. 实验变量设置。
5. 模型训练。
6. 训练过程可视化。
7. 模型测试。
8. 混淆矩阵。
9. 错误样本分析。
10. 实验报告生成。

技术栈：

前端：Vue3 + Vite + Vue Router + Pinia + Element Plus + ECharts + TensorFlow.js。

后端：Flask + Flask-CORS + Flask-SQLAlchemy + SQLite。

初版模型方案：浏览器端 TensorFlow.js + MobileNet 迁移学习。不要优先实现 ResNet50。

页面路由：

1. `/` 首页。
2. `/guide` 实验导学页。
3. `/experiments` 实验任务选择页。
4. `/lab/:experimentId` 实验操作页。
5. `/analysis/:recordId` 结果分析页。
6. `/report/:recordId` 实验报告页。
7. `/gallery` 作品展示页。

开发原则：

1. 每次只完成当前任务要求。
2. 不要自行增加大型平台功能。
3. 不要破坏已有页面和路由。
4. 保持代码结构清晰，组件化实现。
5. 所有功能必须服务虚拟实验主线。
6. 重要逻辑需要添加简短注释。
7. 输出时说明修改了哪些文件、如何运行、如何验收。
