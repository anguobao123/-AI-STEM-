# 阶段验收清单

## 1. 通用验收

- [ ] 项目能启动。
- [ ] 页面不白屏。
- [ ] 控制台无持续红色报错。
- [ ] 当前任务功能可访问。
- [ ] 未引入无关平台功能。
- [ ] 未破坏上一阶段已完成内容。

## 2. 前端原型验收

- [ ] 首页能说明项目定位。
- [ ] 导学页能解释基础概念。
- [ ] 实验选择页有 5 个实验任务。
- [ ] 实验操作页有步骤向导。
- [ ] 分析页有模拟指标、曲线、矩阵。
- [ ] 报告页有完整结构。
- [ ] 作品展示页有卡片列表。

## 3. 上传与配置验收

- [ ] 能创建类别。
- [ ] 能上传图片。
- [ ] 能显示图片预览。
- [ ] 能统计每类样本数量。
- [ ] 样本不足有提示。
- [ ] 能设置训练轮次。
- [ ] 能设置学习率和 batch size。
- [ ] 能保存当前配置。

## 4. 模型训练验收

- [ ] TensorFlow.js 能正常加载。
- [ ] MobileNet 能加载成功。
- [ ] 图片能完成预处理。
- [ ] 能启动训练。
- [ ] 曲线能实时更新。
- [ ] 训练完成后有准确率结果。
- [ ] 训练失败有友好提示。

## 5. 测试与分析验收

- [ ] 能上传测试图片。
- [ ] 能输出预测类别。
- [ ] 能显示置信度。
- [ ] 能选择真实类别。
- [ ] 能记录错误样本。
- [ ] 能计算混淆矩阵。
- [ ] 能生成改进建议。

## 6. 报告验收

- [ ] 报告包含实验目的。
- [ ] 报告包含实验原理。
- [ ] 报告包含数据集说明。
- [ ] 报告包含参数设置。
- [ ] 报告包含结果指标。
- [ ] 报告包含错误样本分析。
- [ ] 报告包含实验结论。
- [ ] 报告可复制或导出。

## 7. 最终演示验收

- [ ] 能在 5 到 10 分钟内完成完整演示。
- [ ] 用户能清楚看到实验变量。
- [ ] 用户能看到训练过程。
- [ ] 用户能看到结果分析。
- [ ] 用户能生成实验报告。
- [ ] 项目整体像虚拟实验，而不是普通管理平台。
