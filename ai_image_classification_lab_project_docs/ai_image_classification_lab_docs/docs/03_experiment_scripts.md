# 03 实验内容脚本文档

## 1. 实验一：认识图像分类

### 页面标题

AI 是如何识别图片的？

### 实验目标

理解图像分类模型会根据图片特征输出类别预测和置信度。

### 页面说明文案

在这个实验中，你可以上传一张图片，系统会使用已有模型对图片进行分类。请注意，模型并不是像人一样理解图片，而是根据训练过程中学到的视觉特征进行概率判断。

### 操作步骤

1. 点击上传图片。
2. 系统显示图片预览。
3. 模型输出预测类别。
4. 系统显示 Top 3 置信度。
5. 学生判断预测是否合理。

### 系统反馈

1. 预测类别。
2. 置信度条形图。
3. 如果置信度较低，提示“模型判断不确定”。

### 思考题

1. 为什么模型会给出多个预测类别？
2. 置信度高是否一定代表预测正确？
3. 什么样的图片更容易被模型识别？

## 2. 实验二：样本数量影响实验

### 页面标题

样本越多，模型一定越好吗？

### 实验目标

探究样本数量对模型准确率和稳定性的影响。

### 操作步骤

1. 创建 2 到 3 个类别。
2. 第一轮每类上传少量图片。
3. 启动训练，保存结果。
4. 第二轮增加每类图片数量。
5. 再次训练，保存结果。
6. 对比两轮准确率、错误样本和置信度。

### 页面提示

建议每类至少上传 10 张图片。如果样本过少，系统可以继续运行，但会提示“实验结果可能不稳定”。

### 观察指标

1. 训练准确率。
2. 验证准确率。
3. 测试图片预测正确率。
4. 错误样本数量。

### 自动分析文案

当样本数量较少时，模型可能只能记住少数图片的特征；当样本数量增加后，模型通常能够学习到更稳定的类别特征。

## 3. 实验三：训练轮次影响实验

### 页面标题

训练越久，模型一定越好吗？

### 实验目标

理解训练轮次对模型学习效果的影响。

### 操作步骤

1. 使用同一数据集。
2. 设置训练轮次为 5。
3. 启动训练并保存结果。
4. 设置训练轮次为 20。
5. 再次训练并保存结果。
6. 对比训练曲线。

### 观察指标

1. loss 是否持续下降。
2. accuracy 是否持续上升。
3. validation accuracy 是否停滞或下降。
4. 测试图片识别是否稳定。

### 自动分析文案

如果训练轮次过少，模型可能还没有充分学习；如果训练轮次过多，训练准确率很高但验证准确率下降，可能出现过拟合。

## 4. 实验四：类别相似度影响实验

### 页面标题

为什么相似图片更容易分错？

### 实验目标

探究类别相似度对模型分类难度的影响。

### 操作步骤

1. 选择差异明显类别进行训练。
2. 查看混淆矩阵。
3. 选择相似类别进行训练。
4. 再次查看混淆矩阵。
5. 对比哪些类别容易混淆。

### 观察指标

1. 类别准确率。
2. 混淆矩阵中错误集中区域。
3. 错误样本图片。

### 自动分析文案

类别外观越相似，模型越难找到稳定区分特征，因此混淆矩阵中更容易出现相互误判。

## 5. 实验五：数据增强影响实验

### 页面标题

让模型见过更多变化

### 实验目标

理解数据增强如何提升模型泛化能力。

### 操作步骤

1. 使用同一数据集。
2. 关闭数据增强训练一次。
3. 开启翻转、裁剪、亮度变化后再次训练。
4. 使用新的测试图片测试。
5. 比较两次结果。

### 观察指标

1. 验证准确率。
2. 测试图片预测结果。
3. 置信度变化。
4. 错误样本数量。

### 自动分析文案

数据增强可以模拟图片角度、位置、亮度等变化，使模型学习到更通用的特征。但过度增强也可能引入不合理样本，影响训练效果。
