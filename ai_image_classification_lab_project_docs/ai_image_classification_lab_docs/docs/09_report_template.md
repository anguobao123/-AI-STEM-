## 6. 自动结论生成规则

### 样本数量实验

如果样本增加后准确率提升：

说明样本数量增加后，模型学习到的类别特征更加稳定，泛化能力有所提升。

如果样本增加后准确率未明显提升：

可能是图片质量、类别相似度或训练参数仍然限制了模型表现。

### 训练轮次实验

如果训练准确率高但验证准确率下降：

可能出现过拟合，模型过度适应训练数据。

如果训练和验证准确率都较低：

可能训练不足、样本数量不足或类别特征不明显。

### 类别相似度实验

如果混淆集中在两个类别之间：

说明这两个类别视觉特征相似，模型难以区分。

### 数据增强实验

如果开启增强后测试效果提升：

说明数据增强提高了模型对新图片变化的适应能力。
