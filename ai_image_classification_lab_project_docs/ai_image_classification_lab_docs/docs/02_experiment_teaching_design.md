# 02 虚拟实验教学设计文档

## 1. 实验总主题

探究影响 AI 图像分类模型效果的关键因素。

## 2. 教学目标

学生完成实验后，应能够：

1. 解释图像分类、样本、标签、训练、验证、推理的基本含义。
2. 理解训练数据对模型表现的影响。
3. 通过训练曲线判断模型训练状态。
4. 通过混淆矩阵分析类别混淆问题。
5. 通过错误样本分析提出模型改进建议。
6. 生成一份结构完整的 AI 图像分类实验报告。

## 3. 知识点清单

1. 图像分类：让模型判断图片属于哪个类别。
2. 样本：用于训练或测试的图片。
3. 标签：图片对应的正确类别。
4. 数据集：由多个样本和标签组成的训练材料。
5. 训练集：用于模型学习的数据。
6. 验证集：用于观察模型泛化效果的数据。
7. 准确率：预测正确的比例。
8. 损失值：模型预测结果与真实结果的差距。
9. 过拟合：模型过度记住训练数据，对新图片表现变差。
10. 混淆矩阵：展示真实类别和预测类别对应关系的分析工具。
11. 数据增强：通过翻转、裁剪、亮度变化等方式增加样本多样性。

## 4. 实验变量设计

| 变量 | 学生操作 | 观察指标 |
|---|---|---|
| 样本数量 | 控制每类图片数量 | 准确率、错误数 |
| 样本质量 | 上传清晰或模糊图片 | 置信度、错误样本 |
| 训练轮次 | 设置 epochs | loss / accuracy 曲线 |
| 数据增强 | 开启或关闭增强 | 验证准确率、泛化效果 |
| 类别相似度 | 选择差异大或相似类别 | 混淆矩阵 |

## 5. 实验任务设计

### 实验一：认识图像分类

目标：理解模型如何对图片进行分类。

任务：上传一张图片，观察预训练模型预测类别和置信度。

重点：模型输出的是概率判断，不是人类式理解。

### 实验二：样本数量影响实验

目标：理解样本数量与模型稳定性的关系。

任务：使用不同样本数量训练模型，对比结果。

变量：每类样本数量。

结论方向：样本太少会导致模型泛化能力不足。

### 实验三：训练轮次影响实验

目标：理解欠拟合和过拟合。

任务：分别设置 5、20、50 轮训练，对比曲线。

变量：训练轮次。

结论方向：训练太少效果不足，训练过多可能过拟合。

### 实验四：类别相似度影响实验

目标：理解任务难度对模型表现的影响。

任务：对比差异大类别和相似类别的分类效果。

变量：类别相似度。

结论方向：类别越相似，越容易被模型混淆。

### 实验五：数据增强影响实验

目标：理解数据增强对泛化能力的作用。

任务：对比开启和关闭数据增强时的模型测试效果。

变量：数据增强开关。

结论方向：合理数据增强可以提升模型对新图片的适应能力。

## 6. 学生操作流程

1. 阅读实验目标。
2. 查看实验原理。
3. 准备或选择数据集。
4. 设置实验变量。
5. 启动训练。
6. 观察训练过程。
7. 上传测试图片。
8. 查看混淆矩阵和错误样本。
9. 填写或确认实验结论。
10. 生成实验报告。

## 7. 评价标准

| 维度 | 标准 |
|---|---|
| 操作完成度 | 是否完成数据上传、训练、测试、报告 |
| 变量控制 | 是否至少调整一个变量进行对比 |
| 结果观察 | 是否查看曲线、准确率、错误样本 |
| 分析能力 | 是否能说明错误原因 |
| 报告质量 | 是否包含目的、过程、结果、结论 |

## 8. 实验结论模板

通过本次实验可以发现，图像分类模型的效果会受到数据集质量、样本数量、训练轮次、类别相似度和数据增强方式的影响。当样本数量不足或类别过于相似时，模型容易出现分类错误。通过增加样本、优化图片质量、合理调整训练轮次和使用数据增强，可以提升模型在新图片上的识别效果。
