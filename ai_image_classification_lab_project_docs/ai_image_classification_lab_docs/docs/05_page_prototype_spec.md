# 05 页面原型说明文档

## 1. 首页

### 页面目标

让用户快速知道项目是什么、能做什么、怎么开始。

### 页面模块

1. Hero 首屏
   - 标题：AI 图像分类虚拟实验室
   - 副标题：从样本到模型，理解 AI 如何学习图像
   - 按钮：开始实验、查看导学
2. 实验流程
   - 准备样本
   - 设置变量
   - 训练模型
   - 测试结果
   - 分析错误
   - 生成报告
3. 项目亮点
   - 零代码训练
   - 可视化曲线
   - 混淆矩阵
   - 自动报告
4. 示例成果
   - 准确率曲线卡片
   - 混淆矩阵卡片
   - 报告截图卡片

## 2. 实验导学页

### 页面目标

用较短内容讲清楚实验前必须理解的概念。

### 页面模块

1. 左侧目录。
2. 右侧知识卡片。
3. 概念小测。
4. 进入实验按钮。

### 关键交互

点击左侧目录切换知识点。完成小测后不强制锁定实验，只给出学习建议。

## 3. 实验任务选择页

### 页面目标

让学生选择实验任务。

### 页面模块

1. 推荐路线提示。
2. 5 个实验任务卡片。
3. 每个卡片包含目标、变量、预计时间、难度。
4. 开始实验按钮。

## 4. 实验操作页

### 页面目标

完成从数据准备到训练测试的完整实验操作。

### 布局

顶部：实验名称、实验目标、当前步骤。

左侧：步骤导航。

右侧：当前步骤操作区。

底部：上一步、下一步、保存本轮实验。

### 步骤

1. 实验说明。
2. 数据集准备。
3. 参数设置。
4. 模型训练。
5. 模型测试。
6. 保存记录。

## 5. 结果分析页

### 页面目标

把训练结果转化为实验分析。

### 页面模块

1. 指标概览。
2. 训练曲线。
3. 混淆矩阵。
4. 错误样本列表。
5. 系统分析建议。
6. 生成报告按钮。

## 6. 实验报告页

### 页面目标

形成可提交成果。

### 页面模块

1. 报告标题。
2. 实验目的。
3. 实验原理。
4. 数据集说明。
5. 参数设置。
6. 结果图表。
7. 错误样本分析。
8. 实验结论。
9. 改进建议。
10. 导出或复制按钮。

## 7. 作品展示页

### 页面目标

展示已完成实验作品。

### 页面模块

1. 筛选栏。
2. 作品卡片。
3. 查看报告按钮。
4. 在线测试按钮。

初版不做评论、收藏、审核。
