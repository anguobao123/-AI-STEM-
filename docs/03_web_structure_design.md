# Web 结构设计

## 页面结构

前端采用 Vue 3 单页应用结构，包含以下页面：

- 首页：说明项目用途、核心能力和推荐实验入口
- 导学页：提供 6 个知识模块，先理解再实验
- 实验列表页：按控制变量组织实验设计
- 实验工作台：采用“数据 - 变量与训练 - 测试与结果”三栏结构
- 分析页：围绕实验问题解释结果
- 报告页：输出标准实验报告结构
- 记录页：以卡片形式展示历史实验记录
- 成果展示页：区分“示例成果”和“我的实验成果”

## 实验工作台结构

`Lab.vue` 顶部展示实验目标、控制变量、保持不变条件和观察指标。

主体采用三栏：

1. 左栏：数据集与类别
2. 中栏：变量设置与模型训练
3. 右栏：单图预测、批量测试与结果预览

底部固定实验记录操作区：

- 保存实验记录
- 查看分析
- 查看报告
- 返回实验列表

## 设计约束

- 主流程不展示 IndexedDB 模型保存、加载和诊断入口
- 不把不稳定的模型持久化能力作为主流程
- 优先展示重要实验信息，不堆调试字段
- 页面需要在桌面端和移动端都保持清晰
